Projects

  • Dots and Numbers

    A logic game inspired from a newspaper game made using JavaScript, HTML and CSS that challenges players to create a loop on the board such that each square has as many edges as the number inside the square. This project was made in the Boilermake IX Hackathon in January 2022.

  • Boilermaker Airlines Graph Project

    A project made using Java that implemented Dijkstra and Kruskal's method to determine the optimal destinations that various flight should depart from and land. This project was made in CS251 : Data Structures and Algorithms in November 2021.

  • ChromeCart

    A chrome extension made using JavaScript, HTML, and CSS that combines a users cart from multiple sites like Amazon, Ebay, Walmart, and more into one compact window. This project was made in the Boilermake VIII Hackathon in January 2021.

  • Song Searcher

    A web applications made using React, JavaScript, HTML, and CSS that allows users to search for songs by inputting the song lyrics and/or tempo. This project was made with the help of a mentor as a part of LaunchPad in Fall 2020.

  • Class Searcher

    A web application made with JavaScript, Java, HTML, and CSS that allows Purdue students to find classes that satisfy multiple science core-requirements with greater ease. This project was made in the Hello World Hackathon in October 2020.

  • Image Manipulator

    I analyzed each pixel from images of various dimensions and applied functions to them to retrieve and manipulate data from neighboring pixels, eventually applying various filters on the image in C. This project was made in Harvard's CS50 course in July 2020.

  • Virtual Assistant

    A program made in Java that allowed users to create a todo list as well as check the weather in various locations, using the AccuWeather API, and all of the major soccer games that were taking place that day using API-FOOTBALL's API. This project was made in KTBytes' CS01B course in July 2018.

Experience

Software Development Engineer Intern - Amazon

Seattle, Washington   •   May 2022 - August 2022

Over the course of 12 weeks, I worked on an AWS Product called AppStream, specifically on their Photon Ops Console, a web-based tool that is used by on-calls to audit and troubleshoot different issues that arise. The current data retrieval process for the Photon Ops Console was working inefficiently as it only stored 3 days worth of data and incurred relatively high costs. I was tasked to create a pipeline that would retrieve data relating to the different streaming instances as well as data logs directly from the data base. I did this by querying the search parameters from the user in the database, filtering the data to remove irrelevant information, and then reformatting and returning the data so that it would be displayed properly.
